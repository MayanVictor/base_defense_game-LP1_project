# Base Defense

C++ Developed Project Using Raylib

## Dependences

- MSYS2 UCRT64
- Raylib 5.5
- GCC 15+

## Compile

g++ the_invasion.cpp game.cpp -o the_invasion.exe -I/ucrt64/include -L/ucrt64/lib -lraylib

## Controls

click with the mouse moves the player.
Aim with the mouse.
Shoot with the Key Q.
